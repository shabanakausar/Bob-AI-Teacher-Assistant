# 🚀 Quick Start Guide - Bob AI Teacher Assistant

Get up and running with Bob in under 5 minutes!

---

## ⚡ Installation (3 Steps)

### Step 1: Open Terminal/Command Prompt
Navigate to the project directory:
```bash
cd bob-teacher-assistant
```

### Step 2: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 3: Run the Application
```bash
python app.py
```

---

## 🌐 Access the Application

Open your web browser and go to:
```
http://localhost:5000
```

---

## 🎯 First Time Usage

### 1. Choose a Mode
Click on one of the four learning modes:
- **MCQ Mode** - Multiple choice questions
- **Short Answer** - Brief explanations
- **Long Answer** - Detailed responses
- **Interactive Quiz** - Adaptive learning

### 2. Configure Your Session
- **Topic**: Type any subject (e.g., "Python", "History", "Math")
- **Difficulty**: Select EASY, MEDIUM, or HARD
- **Questions**: Choose 1-20 questions

### 3. Click "Start Learning"

### 4. Answer Questions
- For MCQ: Click your answer
- For text: Type in the box
- Click "Submit Answer"

### 5. Review Feedback
Bob will show you:
- ✅ If you're correct
- 📚 Explanation
- 💡 Tips for improvement

### 6. View Your Report
After finishing, see your:
- Score and accuracy
- Strengths and weaknesses
- Personalized advice

---

## 🆘 Need Help?

**Application won't start?**
- Make sure Python 3.8+ is installed
- Check if port 5000 is free
- Try: `python --version` to verify Python installation

**Questions not loading?**
- Refresh the page
- Check if the Flask server is running
- Look for errors in the terminal

**Want to stop the server?**
- Press `Ctrl + C` in the terminal
